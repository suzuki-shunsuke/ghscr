# ghscr

Get scripts hosted at GitHub

## Requirements

* go
* curl

## Install

```
$ go get suzuki-shunsuke/ghscr
```

## Usage

```
$ ghscr get <user>/<repo>[/<branch or tag or revision>] <path> [-o <path>] [-O]
$ ghscr get <user>/<repo>/<branch or tag or revision>/<path> [-o <path>] [-O]
```
